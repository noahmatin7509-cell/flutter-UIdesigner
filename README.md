# 🎨 Flutter UI Designer

一个功能强大的 Web 版 Flutter UI 编辑器，支持拖拽操作、实时预览、代码导出等功能。

## ✨ 核心特性

- 🎯 **可视化编辑**：左侧 Widget 库，中间画布，右侧属性面板
- 🎨 **丰富的 Widget 库**：基础控件、布局控件、高级组件
- 📱 **实时预览**：支持手机/Web 屏幕预览
- 💾 **代码导出**：一键生成 Dart 代码
- 🔄 **撤销/重做**：完整的编辑历史
- 💾 **项目保存**：JSON 格式存储
- 🎮 **拖拽支持**：直观的拖拽操作体验

## 📋 Widget 库

### 基础控件
- Text（文本）
- Button（按钮）
- TextField（文本输入）
- Image（图片）
- Icon（图标）

### 布局控件
- Column（列布局）
- Row（行布局）
- Stack（堆栈布局）
- Container（容器）
- Scaffold（脚手架）

### 高级控件
- ListView（列表视图）
- GridView（网格视图）
- TabBar（标签栏）
- Card（卡片）
- AppBar（应用栏）
